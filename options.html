Keka Clock-In — Settings

Configure how the extension finds your Keka session and when it reminds you.

Token source

Open hr.keka.com → DevTools (F12) → Application → Local Storage → hr.keka.com, and find the key that holds your access token. If you're not sure, leave the default — the extension also auto-scans for any key containing "token".
Only used as a last resort, when no hr.keka.com tab is open and nothing is cached. Leave blank for normal use.

Reminder behavior

The reminder never appears once today's clock-in is already marked, regardless of this setting.

Reset

Clears all stored tokens, history, and settings.
